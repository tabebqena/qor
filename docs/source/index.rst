Welcome to kore-lib's documentation!
====================================

`Kore-lib` is a python library for building python web application for the kore_ web framework. The kore framework itself is an:

.. _kore: https://docs.kore.io/4.2.0/


`Easy to use web platform for writing scalable web APIs in C and Python. Its primary goals are security, scalability and allowing rapid development and deployment of such APIs.`

The `kore-lib` built with minimalistic design to provide the most needed building blocks for python `kore` application. The building blocks aren't coupled to each others. `Kore-lib` will not force you to use its design or one of its parts. Most of times, you can use one building block & swap the others with your own.
